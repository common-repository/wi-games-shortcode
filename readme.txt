== Description ==

WI Games Shortcode – allows you to easy and quickly insert any game placed on wigames.net!

== How To ==

After downloading and installation of plug-in for a game insert you will need to add in new (or edited) record shortcode like that:

'[wigames id =" 98"]'

You can learn id of game from game Url on the site wigames.net.

Example:

'http://wigames.net/game/98-double-edged /'

Figures at the beginning of utl are also game id.

In shortcode it is also possible to use the width and height parameters which will help you to adjust the size of game in compliance to the template.

Example:
'[wigames id="134" width="650" height="450"]'

== Installation ==

1. Go to your admin area and select Plugins -> Add new from the menu.
2. Search for "WI Games Shortcode Plugin".
3. Click install.
4. Click activate.

Alternatively you can install the plugin manually.

1. Get an archive with the most recent version of WI-Games. 
2. Uncompress the `wi-games` directory from the archive to your `wp-content/plugins` directory.
3. Activate the plugin in the administration panel.
